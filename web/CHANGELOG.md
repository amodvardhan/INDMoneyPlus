# Changelog

All notable changes to the Portfolio Superapp frontend will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-01-XX

### Added
- Initial release of Portfolio Superapp frontend
- Next.js 14 with App Router
- TypeScript strict mode
- Tailwind CSS with design system
- React Query for data fetching
- Framer Motion for animations
- Recharts for data visualization
- Authentication flow (login/register)
- Dashboard with net worth tracking
- Portfolio holdings view
- Instrument detail pages with price charts
- Rebalance proposal flow
- AI assistant chat overlay
- Design system components (Button, Card, Modal, Table, etc.)
- Unit tests with Jest
- E2E tests with Playwright
- Storybook for component documentation
- Docker setup for development and production
- CI/CD pipeline with GitHub Actions
- Accessibility features (WCAG AA)
- Responsive mobile-first design
- Dark mode support
- Theme system (Light, Dark, OLED, High-Contrast)

### Security
- Secure token storage
- Automatic token refresh
- HTTP-only cookies support (dev mode)
